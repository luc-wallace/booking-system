POST and GET Methods and HTML form: A worked example using PHP

When you submit an HTML form, the data that you entered into the form is sent to a server for processing. There are two HTTP methods that can be used to send this data: POST and GET.

POST method is used when you want to submit sensitive or complex data, like passwords or large amounts of text, to the server. With POST, the data is sent in the HTTP request body, rather than in the URL, so it's not visible to users. POST is the preferred method when you need to modify data on the server or perform an action, such as creating a new record in a database.

GET method, on the other hand, is used when you want to retrieve data from the server. With GET, the data is sent in the URL as query parameters, so it's visible to users. GET is best used when you need to retrieve information, such as a search query or a list of records, from the server.

Now, let's take a look at an example of using POST and GET in an HTML form using PHP. Let's say we have a simple form that allows users to enter their name and email address, and we want to submit this data to a PHP script for processing.

Here's what the HTML form code might look like:

<form action="process_form.php" method="post">
  <label for="name">Name:</label>
  <input type="text" name="name" id="name">

  <label for="email">Email:</label>
  <input type="email" name="email" id="email">

  <button type="submit">Submit</button>
</form>

Using the POST Method

In this example, we're using the POST method to submit the form data to the process_form.php script.

Now, let's take a look at what the PHP script might look like to process this form data:

<?php
if ($_SERVER['REQUEST_METHOD'] === 'POST') {
  $name = $_POST['name'];
  $email = $_POST['email'];

  // Do something with the form data, like store it in a database
}
?>

In this script, we're checking to see if the HTTP request method is POST, and then we're accessing the form data using the $_POST superglobal variable.

Using the GET Method

Now, let's take a look at an example of using GET in an HTML form. Let's say we have a form that allows users to search for products on our website, and we want to use the GET method to retrieve search results from the server.

Here's what the HTML form code might look like:

<form action="search.php" method="get">
  <label for="query">Search:</label>
  <input type="text" name="query" id="query">

  <button type="submit">Search</button>
</form>

In this example, we're using the GET method to submit the form data to the search.php script.

Now, let's take a look at what the PHP script might look like to process this form data:

<?php
if ($_SERVER['REQUEST_METHOD'] === 'GET') {
  $query = $_GET['query'];

  // Search for products that match the query
}
?>

In this script, we're checking to see if the HTTP request method is GET, and then we're accessing the search query using the $_GET superglobal variable.
