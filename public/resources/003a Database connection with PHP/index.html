Connect to a Database with PHP

Here's a basic example of how to connect to a MySQL database using PHP:

Pre-requisites: You have XAMPP running a database. You know that name of the database, the mysql user (for us it will be 'root') and password (for us it is either blank or 'password')

First, you need to create a database connection using the mysqli_connect() function in PHP. Here's an example:

$servername = "localhost"; // likely "localhost"
$username = "username"; // "root"
$password = "password"; // "" blank or "password" or what you set it to be
$dbname = "database_name"; // your DB name

// Create connection
$conn = mysqli_connect($servername, $username, $password, $dbname);

// Check connection
if (!$conn) {
    die("Connection failed: " . mysqli_connect_error());
}

In this example, you need to replace localhost with the hostname of your MySQL server, username with the username you use to connect to MySQL, password with the password for that user, and database_name with the name of the MySQL database you want to connect to.

After the connection is created, you can execute queries on the database using the mysqli_query() function. For example:

$sql = "SELECT * FROM table_name";
$result = mysqli_query($conn, $sql);

if (mysqli_num_rows($result) > 0) {
    // output data of each row
    while($row = mysqli_fetch_assoc($result)) {
        echo "id: " . $row["id"]. " - Name: " . $row["name"]. "<br>";
    }
} else {
    echo "0 results";
}

mysqli_close($conn);

In this example, we're selecting all rows from a table called table_name. We're then using a loop to output the data for each row. Finally, we close the database connection using the mysqli_close() function.
