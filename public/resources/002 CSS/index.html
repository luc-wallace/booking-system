An introduction to CSS

CSS, which stands for Cascading Style Sheets, is a language used to describe the visual style and layout of web pages written in HTML. By using CSS, you can control how web page elements are displayed, such as font size, colour, spacing, and positioning.

Here's a simple example to help you understand how CSS works:

Let's say you have an HTML document with a heading element that looks like this:

<h1>Hello World</h1>

To change the font size and colour of the text in this heading using CSS, you would first create a CSS rule. A CSS rule consists of a selector, which is used to target the HTML element you want to style, and a set of declarations, which specify the style properties you want to apply.

Here's an example CSS rule that targets the h1 element and changes the font size to 24 pixels and the colour to red:

h1 {
  font-size: 24px;
  color: red;
}

To apply this style to your HTML document, you would add this CSS rule to a <style> block in the <head> section of your HTML file, like this:

<!DOCTYPE html>
<html>
  <head>
    <title>My Web Page</title>
    <style>
      h1 {
        font-size: 24px;
        color: red;
      }
    </style>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

When you open this HTML file in a web browser, you will see that the text "Hello World" is displayed in a larger font size and the colour red, as specified in the CSS rule.

External CSS

In addition to using inline in tags or CSS that is stored in the <style> block of an HTML file, you can also create an external CSS file and link to it in your HTML document. This can be a more efficient way to manage your styles if you have multiple pages that share the same styling.

Here's an updated example that shows how to use an external CSS file:

First, create a new file named styles.css and save it in the same directory as your HTML file. In this file, you can define your CSS rules just like you would in an inline <style> block:

h1 {
  font-size: 24px;
  color: red;
}

Next, in your HTML file, add a link to the external CSS file in the <head> section using the <link> element. The href attribute specifies the path to your CSS file, and the rel attribute should be set to "stylesheet":

<!DOCTYPE html>
<html>
  <head>
    <title>My Web Page</title>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

When you open this HTML file in a web browser, you should see the same result as before: the text "Hello World" is displayed in a larger font size and the colour red, as specified in the CSS rule.

Using an external CSS file can help keep your code organised, modularised and make it easier to update your styles across multiple pages. In fact: that is one reason CSS is called 'cascading': make a change in one file and the effects 'cascade' through multiple referring files.
