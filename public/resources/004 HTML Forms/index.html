An Introduction to HTML Forms

In HTML5, forms are used to collect information from users. They allow you to create a user interface with input fields, such as text boxes, checkboxes, radio buttons, and more.

Here's a simple example of an HTML form that asks for a user's name and email address:

<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name"><br>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email"><br>

  <input type="submit" value="Submit">
</form>

In this example, the <form> element encloses the entire form and the </form> tag signals the end of the form. Each input field is enclosed in its own <label> element, which is associated with the input using the for attribute and the id attribute.

The type attribute of each input field determines the type of data that will be collected. For example, the type="text" attribute specifies a simple text input field, while type="email" specifies an email address input field.

Finally, the <input> element with type="submit" creates a submit button that, when clicked, will send the form data to a server-side script for processing.
